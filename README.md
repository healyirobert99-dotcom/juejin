## 项目背景

本系统脱胎于一个**更宏大、但过度复杂**的原版主题主线识别系统（zhuxian-catch）。原系统包含完整的 Point-in-Time 状态引擎、22 个 Python 模块、4328 行生成代码、9 章节日报、完整的失败归因（F1 即时失败 22% / F3 慢退 12% / F4 结构失败 8%）。**但它的回测胜率只有 55%，中位超额几乎为 0。**

在评估原系统的 2328 个 CONFIRMED 事件后，我们发现唯一**通过严格验收**的模块是 **「企稳重估」**——92 个事件，69.83% 胜率，平均超额 3.63%。

因此我们做了**关键决策**：抛弃复杂状态机，聚焦企稳重估逻辑，将其提炼为 **B 方案 4 因子**，仅用 ~420 行核心代码，重新搭建了 v0.6 掘金信号系统。

---

## 回测结论（一句话）

> **B 方案 4 因子宽度回看信号在 16 年全量数据上穿越牛熊：牛市 100% 命中，熊市 78.6% 命中，震荡市 72.4% 命中。可上实盘（小仓位 5-10% 起步 + 进度桶风控）。**

## 完整回测历程

### A 方案——单因子企稳复现

**目标**：验证企稳重估信号在"信号→250 日历史峰值"维度下的有效性。

| 指标 | 值 |
|---|---|
| 总信号数 | 2,619 个 |
| 8% 命中率 | **77.4%** |
| 中位最大收益 | **+20.98%** |
| 负收益占比 | 3.3% |
| 中前期信号（0~50% 进度）占比 | **98.9%** |

**结论**：信号发出时几乎全部在中前期（98.9%），8% 有效门槛命中率 77.4%，验证了行业的有效性。但单因子噪音太大（2619 个信号太多）。

### B 方案——4 因子多因子企稳

**4 个因子**：
1. **行业宽度** ≥ 35%（过去 20 日均）
2. **量比** ≥ 1.0（过去 20 日均）
3. **行业平均 20 日涨幅** > 0（弱转强）
4. **真从弱转强**——过去 60 日多数时间宽度 < 35%，最近 1 日 ≥ 35%

| 指标 | 值 |
|---|---|
| 总信号数 | **542 个**（砍掉 79% 噪音） |
| 8% 命中率 | **73.1%** |
| 中位最大收益 | +21.24% |
| 负收益占比 | 3.5% |

**关键发现**：B 方案砍掉 79% 噪音，但**含金量几乎不变**——绝对收益和命中率与 A 方案相当。说明这 4 个因子的筛选是有效的（去噪音不丢信号）。

### D 方案——核心指标优化

**样本外（OOS）验证 — 最近 Cutoff 后 54 个事件**：
- 100% 命中，中位 +47%

**失败归因**：
- 失败样本：平均 n_stocks = 16 vs 成功 35
- 冷却期反直觉：首次触发 59.4% < 重复触发 82.4%
- **生效规则**：n_stocks < 20 的行业过滤 + 重复触发优先

### D2 方案——4 个 Cutoff 全量 OOS 验证

**这是最关键的验证**——用 4 个历史时间点做 Cutoff，分别验证未来 2.5~11 年的样本外表现：

| Cutoff 时间 | OOS 跨度 | 信号数 | 8% 命中率 | 中位 max_return |
|---|:---:|---:|---:|---:|
| 2014-12-31 | 11 年 | 377 | **81.7%** | +26.83% |
| 2017-12-31 | 8 年 | 318 | **81.1%** | +26.92% |
| 2020-12-31 | 5 年 | 172 | **82.0%** | +24.43% |
| 2023-06-30 | 2.5 年 | 96 | **86.5%** | +34.79% |

**按市场环境分桶**：

| 市场环境 | 信号数 | 命中率 | 评价 |
|:---:|---:|---:|:---|
| 2018 熊市 | 103 | **78.6%** | ✅ 穿越熊市 |
| 2022 震荡 | 29 | **72.4%** | ✅ 穿越震荡 |
| 2023 震荡 | 44 | 59.1% | ⚠️ 唯一衰减 |
| 2024-2025 牛市 | 54 | **100%** | ✅ 完美 |

**结论**：B 方案在熊市（78.6%）和震荡市（72.4%）仍然有效。2023 年衰减是唯一值得关注的异常点。

## v6 影子测试——4 进度桶风控

### 规则（已锁定）

| 桶 | 触发条件 | 止损 | 止盈（减 1/3） |
|---|---|---|---|
| 极早期 | progress < 10% | -18% | +30% |
| 早期 | 10% ≤ progress < 30% | -15% | +30% |
| 中期 | 30% ≤ progress < 50% | -12% | +25% |
| 晚期 | ≥ 50% | -8% | +20% |

**仓位**：首次触发 8%，120 日内重复触发 12%，总仓位上限 30%。

### 影子测试结果（16 个 ETF 回测）

| 版本 | 规则 | 累计收益率 | 胜率 | 评价 |
|:---:|:---|:---:|:---:|:---|
| 裸持有 | 买入不动 | +6.10% | 73% | 高风险 |
| v3 | 15% 机械止损 + 30% 止盈 | +134% | 37.3% | 简单可用 |
| v4 | 20MA 3 日行为止损 | -26% | 29.7% | ❌ A 股震荡市失败 |
| v5 | 加仓 + 20% 总止损 | -1252% | 0% | ❌ 完全失败 |
| **v6** | **4 进度桶分规则** | **+161%** | **40.4%** | **✅ 胜出** |

**为什么 v6 不是简单地设一个止损**：
- 极早期（刚起步）给更大容错空间（-18%）
- 晚期（已涨很多）止损更紧（-8%），因为回撤空间大
- 止盈同样随进度递减（从 +30% 到 +20%）

### 信号工作流

```
stock_data.db (16年日线) ──→ load_raw_daily()
                                   ↓
                           compute_per_stock_indicators()
                                   ↓    计算：60日均线/120日均线/60日最低/量比等
                           compute_industry_daily_metrics()
                                   ↓    聚合：行业宽度、量比、20日均涨幅
                           detect_stabilizing_b()
                                   ↓    4因子筛选
                           mark_repeat_priority()
                                   ↓    标记首次/重复
                           行业信号 + ETF 匹配 → 日报
```

### 实盘监控流

```
用户录入 BUY ──→ live_trades 表 ──→ monitor_all_positions_v6()
                                            ↓
                                    拉取 ETF 日线价格
                                            ↓
                                    计算进度桶 + 浮盈
                                            ↓
                                    止损/止盈判定
                                            ↓
                                    生成告警 + 操作建议
```

## 系统特性

### 信号检测（掘金信号）

| 特性 | 描述 |
|---|---|
| 数据范围 | 16 年 A 股全量（2010-01-01 起） |
| 数据源 | 后复权日线（stock_data.db） |

| 重复优先 | 120 日内重复信号标记，优先建仓 |
| 小行业过滤 | n_stocks < 20 不参与 |

### 实盘监控（v6 进度桶）

| 特性 | 描述 |
|---|---|
| 入场识别 | 标的代码（ETF/个股/行业）+ 买入价 + 金额 |
| 进度计算 | (现价 - 60日最低) / 60日最低 |
| 风控规则 | 4 进度桶，不同止损/止盈 |
| 告警类型 | 止损(红) / 止盈(橙) / 关注(黄) / 正常(绿) |
| 操作建议 | 清仓 / 减仓 1/3 / 关注 / 调整桶 / 持有 |
| 减仓/清仓 | 未平仓列表点选，金额自动按比例预填 |

| 特性 | 描述 |
|---|---|
| ETF 数量 | 98 只（覆盖 15 个大板块） |
| 匹配层数 | 4 层（精确/别名/模糊/子串） |

```bash
git clone https://github.com/healyirobert99-dotcom/juejin.git

pip install -e ".[dev]"
```

```bash
echo "TUSHARE_TOKEN=你的token" > .env
```

## 投资风险声明

⚠️ **重要**：

1. **回测 ≠ 实盘**。73-78% 的 16 年回测命中率不代表未来表现。

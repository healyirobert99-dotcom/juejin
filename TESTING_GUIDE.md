## 4. 设计原则

| 原则 | 体现 |
|---|---|
| **5 秒看完** | 摘要 4 卡 |
| **30 秒决策** | 操作清单 + 持仓监控卡片 |
| **1 分钟研究** | 持仓卡片含止损/止盈价 + 桶状态 + 操作建议 |
| **颜色编码** | 红（止损）/ 橙（止盈）/ 黄（关注）/ 绿（正常）|
| **响应式** | 手机/平板/桌面自适应（480px 自动 2 列）|
| **设计令牌** | CSS 变量统一管理颜色/间距/字体 |

## 5. 关键改进（vs 极简版）

| 维度 | 极简版 | 最终版 |
|---|---|---|
| **摘要卡** | 4 张简单数字 | 4 张带边框色（蓝/红/默认/绿）|
| **操作清单** | 简单列表 | 含宽度/量比核心数据 |
| **持仓卡** | 单行 + 一句话 | **3 行布局**：名称/状态+浮盈 / 桶+日期+现价 / 止损价+止盈价 / 操作建议 |
| **告警** | 简单 | 红色/橙色背景 + 操作建议 chip |
| **规则速查** | 删除 | 4 桶规则卡片（速查用）|
| **响应式** | 无 | 480px 自动 2 列 |
